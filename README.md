# 🎯 Pixel Dart · 像素飞镖

> 像素风横版飞镖投掷手机网页游戏

一款采用像素美术风格的飞镖投掷游戏，专为移动端触摸操作设计。通过点击屏幕投掷飞镖命中靶心，赚取金币解锁技能树，体验不断成长的投掷乐趣。内置刮刮乐彩票小游戏，增添随机惊喜。

## ✨ 特色功能

### 🎮 核心玩法
- **触摸投掷**：点击屏幕即可投掷飞镖，准星自动上下摆动，考验时机把握
- **双层节奏**：准星摆动渐快 + 出手方向独立振荡，形成不可背板的 Lissajous 轨迹
- **2D 靶盘**：飞镖落点由 X（方向摆动）和 Y（准星 + 风偏）共同决定
- **风向系统**：双正弦叠加的平滑风向，顶部 UI 条 + 靶面预测标记，将随机性转化为可预判的技巧要素
- **连击机制**：连续命中积累连击倍率，分数滚雪球式增长；失误可按护盾比例保留部分连击

### 🌳 技能拓扑树
29 个技能节点，四条分支向四角辐射，总费用约 9 万金币：

| 分支 | 方向 | 核心效果 |
|------|------|----------|
| 🎯 **目标** | 左上 | 扩大中心点、环分倍率、磁吸、命运之眼 |
| ⚡ **速度** | 右上 | 冷却缩减、飞镖加速、双发概率、闪电神速 |
| 🔮 **连击** | 左下 | 连击上限、连击护盾、抗风、终极技巧 |
| 🐾 **宠物** | 右下 | 解锁宠物、提升频率/精准/数量、宠物军团 |

> 金币仅来自基础分（控制经济）；连击倍率只放大「分数」用于冲榜，不放大金币产出。

### 🎰 刮刮乐彩票
- 累计获得 500 金币后解锁
- 多档位可选（铜/银/金/钻），越贵潜在回报越高
- 手指涂抹刮开涂层，覆盖率达 50% 自动揭晓
- 按档位独立的幸运值（pity）系统：连续未中提升中奖概率，中奖清零
- 实时统计：投入/返奖/最高单次中奖/未中次数

### 🎨 视听体验
- **纯代码像素渲染**：所有图形由 Canvas 2D 像素级绘制，无外部图片资源
- **纯合成音效**：WebAudio API 实时合成 8-bit 复古音效，零音频文件依赖
- **屏幕震动**：命中靶心触发震屏反馈
- **粒子特效**：命中粒子爆发 + 冲击波扩散
- **飘字反馈**：分数、标签、连击倍率浮动显示
- **宠物伙伴**：解锁后自动投掷，独立音效和特效

### 💾 存档系统
- 金币、分数、技能、彩票统计自动保存至 `localStorage`
- 支持 v1 → v2 存档自动迁移
- 累计获得金币追踪（避免"花光→重新锁定彩票"）

## 🚀 快速开始

### 环境要求
- [Node.js](https://nodejs.org/) ≥ 18

### 安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器（默认端口 4173）
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

Windows 用户也可直接双击 `start.bat`，脚本会自动检测环境并启动。

### 访问方式
- 本地：`http://localhost:4173`
- 局域网 / 手机：`http://<你的IP>:4173`（服务器已开启 `host: true`，监听 `0.0.0.0`）

## 🎯 计分规则

| 区域 | 距离 | 基础分 |
|------|------|--------|
| 🎯 中心 | ≤ centerRadius | 50 × 环分倍率 |
| ⭕ 内环 | ≤ r2 | 25 × 环分倍率 |
| 🔵 中环 | ≤ r3 | 10 |
| ⚪ 外环 | ≤ r4 | 5 |
| ❌ 脱靶 | > r4 | 0 |

- **金币** = 基础分（不受连击倍率影响）
- **分数** = 基础分 × 连击倍率
- 宠物命中奖励按 `petReward` 比例缩放（默认 10%）

## 📱 移动端适配

- viewport 固定，禁止缩放
- 响应式画布：按实际屏幕宽高比动态计算虚拟宽度（480~1300）
- 支持横屏/竖屏一键旋转切换
- PWA 友好：`apple-mobile-web-app-capable` 等 meta 标签
- `ResizeObserver` + `orientationchange` 自动适配
